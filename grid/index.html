Grid

The grid used for Calcite Web is simple, but powerful. Essentially, you create a .container which holds your grid and constrains its proportions, and then use .column-XX classes to layout your page.

The grid is implemented using the viewport-width property, so even if you nest columns, they will always be the same size. A good deal of work has gone into making the grid responsive by default, and the column count is actually different for different screen sizes. To read more about how the grid is responsive, read the responsive section below.
